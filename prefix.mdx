---
title: 'Prefix Configuration'
description: 'Learn about how to configure the server prefix for Crab.'
---

To configure the server prefix for Crab which will be used in the server for **all** prefix commands, please follow the steps below!

<Steps>
<Step title="Run the configuration command!">
Simply run `/config` and an embed will display on your screen.
<Warning>To run this command, you must have **Manage Guild** permission.</Warning>
</Step>
<Step title='Select "Configure Prefix"'>
Once you see the embed, please select the drop-down menu and locate the "Configure Prefix" selection and select it.
</Step>
<Step title='Fill out the form'>
The final step will be to complete the form that will be displayed once your select the "Configure Prefix" selection. It will display an input box where you will type your prefix!
</Step>
</Steps>

All done! Now you can use this prefix in your server to activate the prefix command.
